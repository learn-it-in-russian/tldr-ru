# winget

> Интерфейс командной строки диспетчера пакетов Windows.
> Больше информации: <https://docs.microsoft.com/windows/package-manager/winget>.

- Установить пакет:

`winget install {{package}}`

- Отобразить информацию о пакете:

`winget show {{package}}`

- Найти пакет:

`winget search {{package}}`
