# chage

> Изменяет учётную запись пользователя, дату протухания пароля пользователя.

- Вывести информацию о пароле пользователя:

`chage -l {{username}}`

- Установить дату окончания работы пароля на 10 дней вперёд:

`sudo chage -M {{10}} {{username}}`

- Отключить дату окончания работы пароля:

`sudo chage -M -1 {{username}}`

- Устанавливает дату отключения учётной записи:

`sudo chage -E {{YYYY-MM-DD}}`

- Сменить пароль пользователя при следующем входе:

`sudo chage -d 0`
