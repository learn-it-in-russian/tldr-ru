# envsubst

> Заменить переменные окружения и их значения в shell формате строк.
> Переменные для замены должны быть в формате `${var}` или `$var`.

- Заменить переменные окружения со стандартного потока вывода (stdin) и вывести на стандартный поток вывода (stdout):

`echo '{{$HOME}}' | envsubst`

- Заменить переменные окружения из файла ввода и вывести на консоль:

`envsubst < {{path/to/input_file}}`

- Заменить переменные окружения из файла ввода и вывести в файл:

`envsubst < {{path/to/input_file}} > {{path/to/output_file}}`

- Заменить переменные окружения в файле ввода из списка элементов, разделённого пробелами:

`envsubst '{{$USER $SHELL $HOME}}' < {{path/to/input_file}}`
