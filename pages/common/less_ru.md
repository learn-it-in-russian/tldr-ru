# less

> Открывает файл для интерактивного просмотра, с поддержкой поиска и прокрутки.

- Открыть файл:

`less {{source_file}}`

- Пролистать страницу вверх / вниз:

`<Space> (вниз), b (вверх)`

- Перейти в конец / начало файла:

`G (конец), g (начало)`

- Прямой (от начала в конец) поиск строки (нажмите `n`/`N` для перехода к следующему/предыдущему вхождению):

`/{{something}}`

- Обратный (от конца в начало) поиск строки (нажмите `n`/`N` для перехода к следующему/предыдущему вхождению):

`?{{something}}`

- Открыть текущий файл для редактирования:

`v`

- Выйти:

`q`
