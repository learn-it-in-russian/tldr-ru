# shred

> Перезаписать файлы для безопасного удаления данных (с минимальной возможностью восстановления оригинального содержимого).

- Перезаписать файл:

`shred {{file}}`

- Перезаписать файл, оставляя нули, вместо случайных данных:

`shred --zero {{file}}`

- Перезаписать файл 25 раз:

`shred -n25 {{file}}`

- Перезаписать файл и отметить удалённым:

`shred --remove {{file}}`
