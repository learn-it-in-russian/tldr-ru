# docker ps

> Вывести список Docker контейнеров.

- Напечатать список запущенных docker контейнеров:

`docker ps`

- Вывести все docker контейнеры (работающие и остановленные):

`docker ps --all`

- Выводит последний созданный контейнер (любого состояния):

`docker ps --latest`

- Отфильтровать в выводе контейнеры, содержащие в имени указанную подстроку:

`docker ps --filter="name={{name}}"`

- Отфильтровать контейнеры, которые используют указанный образ в качестве предка:

`docker ps --filter "ancestor={{image}}:{{tag}}"`

- Вывести контейнеры, отфильтрованные по коду завершения (ошибки):

`docker ps --all --filter="exited={{code}}"`

- Вывести отфильтрованный список контейнеров по статусу: создан(created), запущен(running), удалён(removing), остановлен(paused), завершился(exited), мёртв(dead):

`docker ps --filter="status={{status}}"`

- Вывести все контейнеры, которые примонтированы к указанному тому (volume) или имеют том, примонтированный к указанному пути на файловой системе:

`docker ps --filter="volume={{path/to/directory}}" --format "table {{.ID}}\t{{.Image}}\t{{.Names}}\t{{.Mounts}}"`
