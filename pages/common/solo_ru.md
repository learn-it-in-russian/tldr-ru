# solo

> Инструмент для взаимодействия с Solo аппаратными ключами безопасности.

- Выводит список присоединённых Solo:

`solo ls`

- Обновляет текущие присоединённые Solo прошивки до последней версии:

`solo key update`

- Моргнуть подсветкой LED на указанном Solo:

`solo key wink --serial {{serial_number}}`

- Генерирует случайные байты, используя текущие присоединённые Solo, используя безопасный генератор случайных чисел:

`solo key rng raw`

- Мониторит потоковый вывод Solo:

`solo monitor {{path/to/serial_port}}`
