# go env

> Управляет переменными окружения, используемые инструментами Go.

- Показывает все переменные окружения:

`go env`

- Показывает указанную переменную окружения:

`go env {{GOPATH}}`

- Устанавливает значение переменной окружения:

`go env -w {{GOBIN}}={{path/to/directory}}`

- Сбросить значение переменной окружения:

`go env -u {{GOBIN}}`
