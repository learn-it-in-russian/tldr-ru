# docker logs

> Выводит журнал (логи) контейнера.

- Вывести логи указанного контейнера:

`docker logs {{container_name}}`

- Вывести имеющиеся логи и продолжать ждать новые сообщения из указанного контейнера:

`docker logs -f {{container_name}}`

- Напечатать последние 5 строк лога:

`docker logs {{container_name}} --tail {{5}}`

- Вывести на экран логи и добавить к ним время их создания:

`docker logs -t {{container_name}}`

- Напечатать логи за определённый момент времени от запуска контейнера (например: 23m, 10s, 2013-01-02T13:23:37):

`docker logs {{container_name}} --until {{time}}`
