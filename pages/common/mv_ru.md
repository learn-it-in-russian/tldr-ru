# mv

> Переместить или переименовать файлы и директории.

- Переместить файл в указанную локацию:

`mv {{source}} {{target}}`

- Переместить файлы в другую папку, сохраняя их имена:

`mv {{source1}} {{source2}} {{source3}} {{target_directory}}`

- Не спрашивать подтверждение для перезаписи существующих файлов:

`mv -f {{source}} {{target}}`

- Подтверждать действия перед перезаписью существующих файлов, несмотря на права:

`mv -i {{source}} {{target}}`

- Не перезаписывать существующие файлы:

`mv -n {{source}} {{target}}`

- Переместить файлы в подробном режиме, показывает файлы после их перемещения:

`mv -v {{source}} {{target}}`
